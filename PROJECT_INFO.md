# MLAgro React Application

## პროექტის აღწერა
MLAgro არის სოფლის მეურნეობის მართვის თანამედროვე პლატფორმა, რომელიც აწვდის სატელიტურ მონიტორინგს, ამინდის ანალიტიკას და მინდვრების მართვის ინსტრუმენტებს.

## ფუნქციები
- 🔐 **რეგისტრაცია და ავტორიზაცია** - სრული რეგისტრაციის ფორმა ვალიდაციით
- 🛰️ **სატელიტური მონიტორინგი** - რეალურ დროში სატელიტური სურათები
- 🌦️ **ამინდის ანალიტიკა** - ზუსტი პროგნოზი და ისტორიული მონაცემები
- 🗺️ **მინდვრების რუკები** - ციფრული რუკები ზუსტი საზღვრებით
- 📊 **მონაცემთა ანალიტიკა** - დეტალური ანგარიშები და სტატისტიკა
- 👥 **გუნდის მართვა** - თანამშრომლობა და ამოცანების მართვა
- 📱 **მობილური წვდომა** - რესპონსიული დიზაინი ყველა მოწყობილობისთვის

## ტექნოლოგიები
- React 19.2
- TypeScript
- React Router
- Redux Toolkit
- React Query
- Axios
- SCSS
- Vite

## დაყენება და გაშვება

### Prerequisites
- Node.js (v16+)
- npm ან yarn

### დაყენება
```bash
# დაყენება
npm install

# განვითარების სერვერი
npm run dev

# პროდაქშენ build
npm run build

# პროდაქშენ preview
npm run preview
```

## პროექტის სტრუქტურა
```
src/
├── api/              # API კლიენტი და კონფიგურაცია

│   ├── common/      # საერთო კომპონენტები
│   └── layout/      # Layout კომპონენტები
├── contexts/        # React კონტექსტები
├── hooks/           # Custom hooks
├── layouts/         # Page layouts
├── pages/           # გვერდები
│   ├── admin/       # ადმინ პანელის გვერდები
│   ├── company/     # კომპანიის გვერდები
│   ├── satellite/   # სატელიტური გვერდები
│   ├── support/     # მხარდაჭერის გვერდები
│   └── weather/     # ამინდის გვერდები
├── router/          # რაუტინგი
├── services/        # სერვისები
├── styles/          # გლობალური სტილები
├── types/           # TypeScript ტიპები
└── utils/           # დამხმარე ფუნქციები
```

## API კონფიგურაცია
API URL-ის კონფიგურაცია `.env` ფაილში:
```
VITE_API_URL=http://localhost:3000/api
```

## მარშრუტები
- `/` - მთავარი გვერდი
- `/login` - შესვლა
- `/register` - რეგისტრაცია
- `/app/main-menu` - მთავარი მენიუ (ავტორიზაციის შემდეგ)
- `/app/dashboard` - სამუშაო პანელი
- `/app/electronic-field-map` - მინდვრების რუკა
- `/app/my-fields` - ჩემი მინდვრები
- `/app/weather` - ამინდი
- `/app/satellite-data` - სატელიტური მონაცემები
- `/app/company` - კომპანია
- `/app/admin` - ადმინ პანელი (მხოლოდ ადმინისთვის)

## ინტერფეისი
ყველა ინტერფეისი ქართულ ენაზეა გადათარგმნილი, მათ შორის:
- ნავიგაციის მენიუ
- ავტორიზაციის და რეგისტრაციის ფორმები
- შეცდომების შეტყობინებები
- სისტემური შეტყობინებები

## განვითარება
პროექტი იყენებს:
- ESLint კოდის ხარისხის შემოწმებისთვის
- TypeScript ტიპების უსაფრთხოებისთვის
- Vite სწრაფი HMR-ისთვის

## Backend ინტეგრაცია
აპლიკაცია მზადაა ინტეგრაციისთვის `mlagronode` backend-თან:
1. გაუშვით backend სერვერი პორტზე 3000
2. API კლიენტი ავტომატურად დაუკავშირდება `http://localhost:3000/api`

## მთავარი ფუნქციები

✅ დაცული მარშრუტები და ადმინ პანელი
✅ რესპონსიული დიზაინი
✅ ქართული ინტერფეისი
✅ Loading და Error სტეიტები
✅ API interceptors ავტომატური error handling-ისთვის
